# 开始


## 服务器环境

* php >= 5.4
* php extensions:
  * mcrypt
  * gd2
  * lib-xml with DOM support
* mod_rewrite if using Apache

> 注意： 从php5.5开始，一些系统发行版可能要求你手动安装php json扩展名。当使用ubuntu时，这可以通过`apt-get install php5-json`

## 软件要求
* Apache>=2.4
* MySQL >= 5.0
* Composer

## 支持的浏览器
* google chrome
* mozilla firefox
* webkit-based browsers
* Internet Explorer 9+
